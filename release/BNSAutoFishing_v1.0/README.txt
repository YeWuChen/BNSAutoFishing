使用說明

* 初次使用前 *
1. 截圖釣魚按鈕(上釣時)，並儲存到與釣魚程式同一目錄下
   (請參考檔案包內的fishing.png)

* 其後每次使用時 *
1. 以系統管理員身份開啟程式
2. （如沒有啟動釣魚狀態檢查）遊戲客戶端內手動下一次魚餌

* 注意 *
1. 若遊戲視窗解析度有更改過，釣魚按鈕可能需要重新截圖，才能正常運作

自定義配置

程式運行時會搜尋config.ini並讀入自定義配置
若找不到config.ini則自動載入為默認值
格式請參考檔案包內的config.ini

屬性名稱: baitKey
屬性內容: 魚餌位置
默認值: 5
允許值: [5, 6, 7, 8]
描述: 根據你魚餌放置在道具欄上的位置修改

屬性名稱: captureImg
屬性內容: 釣魚按鈕截圖名稱
默認值: fishing.png
描述: 根據你的釣魚按鈕截圖檔案名稱修改

屬性名稱: interval
屬性內容: 刷新時間間距(秒)
默認值: 1
描述: 每次搜尋遊戲視窗的時間間距，值越小程式需要越多資源，建議值為0.5~1.5

屬性名稱: dragDelay
屬性內容: 收竿延遲(秒)
默認值: 0.5
描述: 搜尋到上釣按鈕後收竿的時間間距，值太小有可能導致釣魚失敗，建議值為0.1~1

屬性名稱: threshold
屬性內容: 最低近似度
默認值: 0.8
描述: 搜尋遊戲視窗畫面跟上釣按鈕截圖的匹配閾值，建議值為0.7~0.9

屬性名稱: enableStopCheck
屬性內容: 啟動釣魚狀態檢查
默認值: False
允許值: [True, False]
描述: 啟動後，若連續一定時間內(參考下一項設定)沒有搜尋到上釣按鈕，將自動下魚餌
注意: 啟動此功能的話，請檢查道具欄位置及確定你正在釣魚，否則程式有可能消耗到錯誤的道具

屬性名稱: stopCheckInterval
屬性內容: 釣魚狀態檢查間距(秒)
默認值: 40
描述: 啟動上一項設定才有效。若超過此設定的時間內，連續沒有搜尋到上釣按鈕，將自動下魚餌，建議值為30~60

屬性名稱: showDetails
屬性內容: 啟動釣魚狀態檢查
默認值: False
允許值: [True, False]
描述: 啟動後，程式將顯示更多訊息（包括釣魚程式參數、未發現上釣按鈕、已發現上釣按鈕等）
